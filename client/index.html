Request Header Parser Microservice

  • 1) Going to the path https://wanjongkim-request-header.herokuapp.com/api/whoami you can receive information about the your browser in JSON format
  • 2) The information included in the JSON will be the IP address, language, and software of the browser

Example: https://wanjongkim-request-header.herokuapp.com/api/whoami

Returns: {"ipaddress":"68.172.233.201","language":"en-US,en;q=0.8","software":"Mozilla/5.0 (Windows NT 6.1) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/51.0.2704.103 Safari/537.36"}
